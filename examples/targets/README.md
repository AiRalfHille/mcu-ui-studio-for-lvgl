# Target Examples

Dieses Verzeichnis sammelt Referenzintegrationen fuer konkrete Zielsysteme.

Im Unterschied zu den normalen Editor-Beispielen liegen hier nicht nur
`.lvglproj`-Dateien, sondern komplette Laufzeitprojekte mit handgeschriebenem
Code, Build-Dateien und Projekt-Dokumentation.

Aktueller Inhalt:

- `portal`
  Referenzintegration fuer das Editor-Beispiel `examples/portal` auf ESP-IDF
  mit `RTOS-Messages`, `lv_conf`-Overlay, Projekt-Theme und generierten
  `ui_start_*`-Dateien.

- `kachel`
  ESP-IDF-Referenz fuer das Editor-Beispiel `examples/kachel` mit
  `Standard`-Template, Dispatcher-Pfad und Update-Funktionen.

- `widgets`
  schlankes ESP-IDF-Displayprojekt fuer das Editor-Beispiel `examples/widgets`
  ohne Event- und Update-Demo.
