# GitHub-Dokumentation

Dieses Verzeichnis enthaelt eine MkDocs-Material-Dokumentation rund um:

- GitHub als Heimat fuer das Projekt
- Open-Source-Grundlagen
- empfohlene Repository-Dateien
- Lizenz, Issues, Wiki und Einstellungen
- projektbezogene Beschreibung von MCU UI Studio for LVGL

## Lokaler Start

```bash
cd GitHub
mkdocs serve
```

## Statischer Build

```bash
cd GitHub
mkdocs build
```
