# Contributing

Thank you for your interest in `MCU UI Studio for LVGL`.

At the moment this project is maintained in a controlled way. Feedback is
welcome, but direct code contributions from third parties are currently not the
default workflow.

## What To Use

Please use GitHub Issues for:

- bugs and regressions
- feature requests
- documentation improvements
- questions about current behavior

## Pull Requests

Pull requests from third parties are currently not part of the regular project
workflow.

If you want to propose a larger change, please open an issue first so scope,
direction, and integration impact can be clarified before implementation work
starts.

## Good Issue Reports

Helpful issue reports usually include:

- a short and clear title
- the affected area
- steps to reproduce
- expected behavior
- actual behavior
- screenshots or logs when useful
- platform details when relevant, for example macOS, Windows, or Linux

## Scope Of Contributions

This repository combines:

- the editor itself
- generated output examples
- target integration examples
- user and project documentation

When reporting changes or ideas, please make clear which area is affected.

## Notes For Future Contributors

If external code contributions are opened up later, the expected direction will
be:

- small and focused changes
- no unrelated refactoring
- documentation updated when behavior changes
- generated code and handwritten runtime code kept clearly separated

## Security

Please do not report security-sensitive issues publicly in a normal issue.
Use the process described in [SECURITY.md](SECURITY.md) once that file is
available.
