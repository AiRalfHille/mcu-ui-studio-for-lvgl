# Security

## Reporting

Please do not report security-sensitive issues through public GitHub issues.

For now, security problems should be reported privately to the project
maintainers.

## Scope

This process applies to:

- the desktop application
- generated code paths
- preview and simulator integration
- example target integrations

## Notes

The project is still under active development. Security-related fixes may first
land in the private working state before they appear in a public repository
snapshot.
