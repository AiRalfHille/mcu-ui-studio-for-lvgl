# Changelog

German version: [CHANGELOG.de.md](CHANGELOG.de.md)

This file tracks notable project changes once versioned releases are prepared.

## v0.1.0

- First public GitHub release of the repository, handbook, and desktop
  packages.
- Windows x64 package is available as a direct desktop download.
- macOS ARM64 package is currently delivered as an unsigned app bundle and
  should be started through `Start MCU UI Studio.command` after unpacking.
- On Windows the handbook currently opens in the external browser because the
  embedded WebView path is not stable enough yet.
