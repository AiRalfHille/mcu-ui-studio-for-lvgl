# Lizenz

MCU UI Studio for LVGL steht unter der **MIT-Lizenz**.

## Kurz gesagt

Die MIT-Lizenz erlaubt es, das Projekt sehr frei zu nutzen.

Dazu gehören insbesondere:

- private Nutzung
- geschäftliche Nutzung
- Veränderung des Quellcodes
- Weitergabe
- Einbindung in andere Projekte

Voraussetzung ist, dass der Urheberrechtsvermerk und der Lizenzhinweis
erhalten bleiben.

## Gewährleistung und Haftung

Das Projekt wird unter der MIT-Lizenz **ohne Gewährleistung** bereitgestellt.
Das bedeutet insbesondere:

- keine Zusicherung, dass die Software fehlerfrei ist
- keine Haftung für Schäden, die aus der Nutzung entstehen
- keine Garantie für Eignung zu einem bestimmten Zweck

## Erweiterungen und Plugins

Die MIT-Lizenz dieser Anwendung bedeutet nicht automatisch, dass jede spätere
Erweiterung ebenfalls unter MIT stehen muss.

Wenn zukünftig eine Plugin- oder Modulschnittstelle entsteht, können
Erweiterungen grundsätzlich auch unter einer anderen Lizenz veröffentlicht
werden, sofern sie klar als eigenständige Module oder separate Komponenten
behandelt werden.

Wichtig ist dabei:

- die Lizenz des Hauptprojekts bleibt davon unberührt
- zusätzliche Module müssen ihre eigene Lizenz klar ausweisen
- Drittanbieter- oder Zusatzmodule sollten nicht stillschweigend als Teil der
  MIT-Lizenz des Hauptprojekts erscheinen
